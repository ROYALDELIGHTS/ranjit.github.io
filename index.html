IPC Insight

IPC Insight is an application designed to predict relevant IPC (Indian Penal Code) sections based on crime narratives using vector embeddings. Developed with Python, Flask, and a web frontend stack (HTML, CSS, JavaScript), it utilizes Chroma DB for storing IPC section embeddings and MongoDB for user management. The application matches crime descriptions with IPC sections to determine the most applicable legal codes

Technologies: html,css,bootstrap,python,Flask
